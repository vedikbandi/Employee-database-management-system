# 👨‍💼 Employee Database Management System

A responsive **Employee Database Management System** built using **HTML, CSS, and Vanilla JavaScript**. This project allows users to manage employee records with an intuitive interface by adding, viewing, selecting, and deleting employees.

> ⚠️ This is a **frontend-only** application. Employee data is stored only in browser memory during runtime and is **not persisted** to any backend or database.

# 🚀 Features

- ✅ View all employees
- ✅ Display employee details
- ✅ Add new employees
- ✅ Delete employees
- ✅ Highlight selected employee
- ✅ Calculate employee age automatically from DOB
- ✅ Default profile image support
- ✅ Responsive UI
- ✅ Handles empty employee database gracefully
- ✅ No page reload required

# ⚙️ How to Run

Open the project using **VS Code** and install the **Live Server** extension.

Right-click on **index.html**

Select

```
Open with Live Server
```

The application will open automatically in your browser.

---

# 📋 Employee Information

Each employee contains the following details:

- Employee ID
- First Name
- Last Name
- Profile Image
- Email Address
- Contact Number
- Salary
- Address
- Date of Birth
- Age

---

# ➕ Add Employee

Click the **Add Employee** button and enter:

- First Name
- Last Name
- Email
- Contact Number
- Salary
- Address
- Date of Birth
- Image URL (Optional)

If no image is provided, the application automatically uses:

```
gf.png
```

---

# 🗑 Delete Employee

Click the ❌ icon beside an employee to remove them from the list.

> Note: Deleting an employee only removes them from the current browser session.

---

# 📌 Important Notes

This project is **frontend only**.

- New employees are **not saved** permanently.
- Deleted employees reappear after refreshing the page.
- Data is loaded from **data.json** every time the application starts.
